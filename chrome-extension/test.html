  • ✅ 所有插件文件已创建

  • ⏳ 在YouTube页面测试功能

🔧 插件配置步骤

  1. 安装插件
    • 打开 chrome://extensions/
    • 开启"开发者模式"
    • 点击"加载已解压的扩展程序"
    • 选择插件文件夹
  2. 配置API
    • 点击插件图标打开设置
    • 填入OpenAI API信息
    • 选择目标语言
    • 保存设置

🎯 测试建议

推荐测试视频： 选择有英文字幕的技术讲解或教程视频

测试步骤：

  1. 打开YouTube视频（确保有英文字幕）
  2. 开启字幕（CC按钮）
  3. 观察是否出现双语字幕
  4. 检查翻译质量和同步性

预期效果：

  • 原文字幕显示在上方（白色）
  • 翻译字幕显示在下方（黄色）
  • 背景为半透明黑色
  • 位置居中显示

🐛 故障排除

常见问题：
  • 插件无反应：检查API密钥、刷新页面
  • 无法加载插件：检查manifest.json语法
  • 翻译失败：检查网络连接和API额度
  • 字幕不显示：确保YouTube字幕已开启
调试提示： 按F12打开开发者工具，查看Console标签页的错误信息

📁 文件检查

确保以下文件都已创建：

  • ✅ manifest.json - 插件配置
  • ✅ content.js - 核心功能脚本
  • ✅ popup.html - 设置界面
  • ✅ popup.js - 设置逻辑
  • ✅ style.css - 样式文件
  • ✅ README.md - 使用说明
